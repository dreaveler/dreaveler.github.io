# 1. 相机
图像形成是视觉感知的基础过程，涉及光线、成像设备和物体表面之间的相互作用。
先来假设sensor直接接受来自世界的光线 那么sensor上的每一个pixel都会接收到来自所有物体的光，从而会产生混合，从而得到的像是混乱模糊的。
### 1.1 针孔相机模型
针孔相机是最简单的成像模型，光线通过一个小孔直接投影到成像平面上。
- 针孔相机产生的图像是倒置的
- 小孔越小，图像越清晰但越暗
![](/images/notes/计算机视觉/ec5ca1d2a6d5-01.png)
物理相机模型与数学相机模型的对比
实际上 物体应成倒像，焦点位于像平面与实际物体中间   但实际为了好算，一般把相平面对称到焦点对侧与实际物体同方向
- 成像公式：1/f = 1/u + 1/v（f为焦距，u为物距，v为像距）
### 1.2 透镜成像
透镜通过折射光线来聚焦，形成图像，它的主要作用是让进光量变大使画面变亮
- 薄透镜公式同样适用：1/f = 1/u + 1/v
- 放大率 = v/u（像高与物高之比）
- 透镜系统存在多种像差：球差、色差、散光、畸变等
### 1.3 由世界坐标系转到图像的过程
- 世界坐标系转为相机坐标系
- 相机坐标系转为像坐标系
### 1.4 相机外参
外参描述相机在世界坐标系中的位置和方向。
- 旋转矩阵R：描述相机坐标系相对于世界坐标系的旋转
- 平移向量t：描述相机光心在世界坐标系中的位置
![](/images/notes/计算机视觉/ec5ca1d2a6d5-02.png)
注意这里是齐次坐标系第一次出现  它的好处就是可以让坐标变换转为矩阵乘
### 1.5 相机内参
内参描述相机的内部光学特性。
- 焦距(focal length)：影响视场角和放大倍率
- 主点(principal point)：光轴与成像平面的交点
- 像素尺寸：决定分辨率
- 畸变系数：描述镜头畸变程度
![](/images/notes/计算机视觉/ec5ca1d2a6d5-03.png)
起始是 $(x_c,y_c,z_c)$ 是相机坐标系下的坐标  而除以 $z_c$ (深度)得到归一化后的坐标，这一过程可由上面的演示图得出  紧接着是乘相机的内参坐标系  f为焦距  $c_x,c_y$是主点在像素坐标系下的offset，最右边的是决定最后的投影是透视投影还是正交投影，如果圈黄的位置是1是透视投影即与深度有关，而是0则是正交投影即与深度无关。
处理过程：归一化后的坐标×Intrincis矩阵×Projection矩阵即可得到像平面下的坐标。
正交投影视光为垂直像平面射入的，一般来说超长焦镜头会是这种效果，而透视则视光会聚于一点，一般的短焦镜头是这种效果。
### 1.6 消失点与消失线
投影的过程不是一个线性变换
点投影到点 线投影到线
下面进行证明 ：
对于一个3D直线来说 我们可以表示为
$$
\mathbf{X}(t)=\mathbf{X}_0+t\,\mathbf{d},\qquad
\mathbf{X}(t)=
\begin{pmatrix}
X(t)\\
Y(t)\\
Z(t)\\
1
\end{pmatrix}
$$
对于一个投影矩阵 可以表示为：
$$
\mathbf{P}=
\begin{pmatrix}
p_{11}&p_{12}&p_{13}&p_{14}\\
p_{21}&p_{22}&p_{23}&p_{24}\\
p_{31}&p_{32}&p_{33}&p_{34}
\end{pmatrix}
$$
那么对于投影结束后的齐次坐标 可以表示为
$$

\mathbf{x}(t)=\mathbf{P}\mathbf{X}_0+t\,\mathbf{P}\mathbf{d}
=
\begin{pmatrix}
x_0+t\,d'_x\\
y_0+t\,d'_y\\
w_0+t\,d'_w
\end{pmatrix}
$$
最后得到欧式平面下的坐标为：
$$
x_{euclidean}(t) = (x_0 + t,d'_x) / (w_0 + t,d'_w)  y_{euclidean}(t) = (y_0 + t,d'_y) / (w_0 + t,d'_w)
$$
消去t后可以得到：
$$
Ax_{euclidean} + By_{euclidean} + C = 0
$$
这就是像空间中的一般方程
这个变换是不保角的
平行线最后会相交于一点 这个点称之为消失点
而一个平面上的平行线投影过后的消失点连接成的线就是消失线


一个像平面上的直线方程的点 我们可以表示为
$$
\mathbf{X}_t=
\begin{pmatrix}
X_0+tD_1\\
Y_0+tD_2\\
Z_0+tD_3\\
1
\end{pmatrix}
\;\;\cong\;\;
\begin{pmatrix}
X_0/t + D_1\\
Y_0/t + D_2\\
Z_0/t + D_3\\
1/t
\end{pmatrix}
$$

而无穷远处的点  也就是t趋于正无穷
$$
\mathbf{X}_\infty=
\begin{pmatrix}
D_1\\
D_2\\
D_3\\
0
\end{pmatrix}
$$
  这就是无穷远点的表示  而这条直线对应的消失点也就是 $PX_{\infty}$ 从这里可以看出所有有相同方向向量的线拥有同一个消失点
需要注意的是 这里是齐次坐标系的第二个好处 即可以表示无穷远点
# 2. Lens
作为透镜，它的作用是捕捉更多的光  它抽象得到的就是一个小孔相机
![](/images/notes/计算机视觉/ec5ca1d2a6d5-04.png)

# 3. Depth of Field(DOF) 景深
我看到这个想到的就是相机的光圈带来的虚化，大光圈带来强虚化。
DOF就是画面中可接受的清楚的最近物体到最远物体的距离。
光圈大，进光量大，则光线经过透镜后的相交点更不宜落于传感器上，光圈小，更易落入传感器，但进光量小，需要提升iso，带来更多噪点
# 4. Field of View(FOV)
FOV是相机能够获取的视野范围
![](/images/notes/计算机视觉/ec5ca1d2a6d5-05.png)
$$
\tan(\alpha/2) = (d/2)/z^{'}
$$
d是传感器的大小   $z{'}  = (1+m)F$   对于焦点可以到达无穷远处的， $z{'} = F$
这里就让人回忆起长焦与短焦镜头的区别
# 5. Lens Aberrations  镜头畸变
由广角镜头引起的  一般见于照片边缘
## 5.1 畸变
### 5.1.1 径向畸变（Radial Distortion）
- 归一化坐标：从相机坐标除以深度得到 $(x_n, y_n),\quad r^2 = x_n^2 + y_n^2$
- 畸变缩放因子： $s(r) = 1 + κ1 r^2 + κ2 r^4 (+ κ3 r^6 …)$
- 畸变后坐标： $x_d = x_n s(r),\quad y_d = y_n s(r)$
- 畸变后再乘焦距加上offset得到新坐标
- 直观：越靠近边缘 r 越大，缩放越明显。κ1 \< 0 常见“桶形”，κ1 \> 0 常见“枕形”。 桶形见于短焦
![](/images/notes/计算机视觉/ec5ca1d2a6d5-06.png)
蓝色镜头显示了畸变的大小与方向
### 5.1.2 标定与去畸变
- 标定：拍摄棋盘格或圆点靶，非线性最小二乘同时估计内参 $K={f_x,f_y,c_x,c_y}$与畸变 ${κ1,κ2,(κ3),p1,p2}$，最小化重投影误差。
- 去畸变（反畸变）：给定像素 $(u,v)$ → 减主点除焦距得 $(x,y)$。迭代求未畸变 $(x_{undist},y_{undist})$，使其经畸变模型后逼近 (x,y)。
- 常用实现：OpenCV 的 calibrateCamera、undistort、initUndistortRectifyMap。
### 5.1.3 常见现象
- 广角端常见桶形（κ1\<0），长焦端常见枕形（κ1\>0）。
- 采集标定图时要覆盖全视场并包含边缘区域。
- 若只做轻量近似，κ1 一个参数即可显著改善；精确应用再加入 κ2, κ3 与切向项。
## 5.2 色差
光透过透镜由于折射率不同不同颜色的光会在不同的距离处成像，导致模糊与色差，为了减少色差以及其它像差，大多数镜头都是同不同玻璃材料以及镀膜组成的复合镜组
## 5.3 晕影
晕影是越靠近图像边缘亮度越容易下降的现象
机械晕影：光束中被遮挡的部分无法到达像平面，而导致边缘变暗
# 6. 颜色
## 6.1 人眼
人眼有虹膜，对焦通过改变透镜的形状 当人眼正确对焦时 外界物体的光就会在视网膜上成像。视网膜上有杆细胞和视锥细胞。
杆细胞
- 数量多
- 只是灰度的感受 与色彩无关
- 高敏感型 可以在夜晚工作
视锥细胞
- 数量较少
- 需要在中等以及强亮度下工作
- 有三种视锥细胞用来感受不同波长的光(420-440,530-540，560-580nm）
## 6.2 CIE RGB
由于人眼视锥细胞的分类，我们用三种特定的颜色来对应人眼视锥细胞感受的颜色
即使用Red Green Blue 来混合颜色
# 7. 图像传感器
![](/images/notes/计算机视觉/ec5ca1d2a6d5-07.png)
## 7.1 shutter 快门
快门位于sensor之前，大多数数字相机使用机械和电子快门 决定sensor的感光时间 即进光量
## 7.2 sensor
- CCD传感器：电荷耦合器件，具有高质量图像但功耗较高
- CMOS传感器：互补金属氧化物半导体，功耗低，集成度高
## 7.3 颜色滤波阵列
![](/images/notes/计算机视觉/ec5ca1d2a6d5-08.png)
为了测量颜色，像素被排列为彩色阵列  每个像素缺失的颜色分量由邻域像素插值得到 (去马赛克)
Bayer RGB排列。每个2x2子马赛克包含2个绿色、1个蓝色和1个红色滤色片，每个滤色片覆盖一个像素传感器，绿色更多是因为人眼的特性。
![](/images/notes/计算机视觉/ec5ca1d2a6d5-09.png)
每个像素按照自身光谱灵敏度对入射光谱进行积分  相机厂商会提供传感器的光谱响应曲线
## 7.4 Gamma 校正
![](/images/notes/计算机视觉/ec5ca1d2a6d5-10.png)
人眼对亮度的感受能力不是线性的  在较暗区域对亮度差异更加敏感 在离散化前，先对强度或颜色做非线性变换 并在读取时把这种变换还原是有益的
