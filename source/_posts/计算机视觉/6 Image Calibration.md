# 3D-Vision
3D视觉的目的是从图片重建三维结构
而其关键的困难点是单视角带来的模糊不清，对于一张照片，许多3D的点可能被投影到二维上的同一个点
解决方式有：使用传感器发射light  使用两个不同位置的照相机生成全景  多视角几何 移动照相机找到关联  由不同阴影下的照片进行重建  从该数据中学习(network)
# Camera calibration
## Review of camera parameters
我们可以回忆 相机具有Intrinsic parameters 以及 Extrinsic parameters
一个2D的点 是由一个3D的点的齐次坐标 左乘内参矩阵 投影矩阵 外参矩阵 可以得到2D的点的齐次坐标
外参矩阵包含rotation以及translation
内参矩阵包括焦距 比例 扭曲 主点等
![](/images/notes/计算机视觉/f2dc40282aa3-01.png)
## Camera calibration
为了求解相机的内参矩阵  我们需要进行相机标定
一个做法就是对于棋盘格上的n个点 我们直到它的3D坐标以及照片上的2D坐标
由于可以将所有的矩阵乘在一起得到一个(3,4)的矩阵 我们可以将其表示为P
而可以计算得到x与P、X的关系  这时一种线性方法
![](/images/notes/计算机视觉/f2dc40282aa3-02.png)
我们可以回忆计算最小二乘解拟合P  那么通过6个不共线的点 我们可以计算出P
通过这种方式得到P是很快的 但它不能直接告诉我们相机参数的信息 而相机参数应该表示为$K[R|t]$
我们可以发现K是一个上三角阵 而R是一个正交阵  那么我们可以回忆线性代数中的RQ分解取得到K,R,t
### QR分解
![](/images/notes/计算机视觉/f2dc40282aa3-03.png)
那么RQ分解是求A的行向量组的正交坐标系  由最后一行开始
我们可以对P的左边三列得到的子阵进行RQ分解 得到K R，然后再用P和K计算t


在实际操作中 非线性的相机标定是preferred 用非线性最小二乘 即用线性计算得到的P并分解得到粗略的K R t后，将X,Y投影为x,y  然后最小化x,y与x',y'的L2
这样可以精细化畸变 阴影等更细小的参数
# Triangulation
在已知相机矩阵的情况下  对于两张或更多的照片，在给出一个3D点的投影坐标后，计算出这个点的3D坐标
其实清楚的思路就是由于有投影坐标与相机矩阵，我们可以将相机与投影点连线并延申  两条线的交点就是X  但由于noise等种种原因 两条线不会相交的如此精确
![](/images/notes/计算机视觉/f2dc40282aa3-04.png)
那么有以下三种方法：
- 几何方法：对于两条光线 得到距离最短线段连接处 取这个线段的中点为X
- 非线性方法：找到使得 $||proj(P_1X)-x_1||^2_2 + ||proj(P_2X)-x_2||^2_2$ 最小的点
- 线性优化：由于我们可以将向量叉乘写为矩阵形式 因此 $x_1 = P_1 X$ 等价于 $x_1 \times P_1X = 0$ 而又可以写为 $x_{1[\times]}P_1X = 0$  那么对于两个方程我们有三个未知数，可以使用有限制的最小二乘
	- 我们一般使用(X,Y,Z,W)作为X  使用 $||X||^2 = 1$作为限制 这样可以使用奇异值分解进行计算
# 有消失点的相机标定
假如3D点的世界坐标系下的坐标不可知，那么在特定情形下，我们可以使用消失点
对于消失点 我们可以利用它连出世界坐标系下的一条线
复习：消失点在齐次坐标系中可以用(X,Y,Z,0)来表示
我们考虑一张图片 它有三个正交的消失点  其中v1 v2是有限制的  v3是无限制的
我们可以通过这三个方向构建出世界坐标系 由于我们将这三个消失点视作单位方向上的点  我们可以对它列出方程
![](/images/notes/计算机视觉/f2dc40282aa3-05.png)
我们会发现其实还可以列出一个世界原点的投影  也就是P(0 0 0 0) = 0
这样就是拥有了四个方程 但对于解出P是不够的  我们需要更多的限制
我们可以首先考虑将P表示为K\[R\|t\]的形式   vi = K\[R\|t\]\[ei 0\]
那么又可以得到ei用vi与参数矩阵表示出来的结果  又因为ei ej是正交的  因此$ei^Tej = 0$  这样对于三组坐标轴 我们可以得到三个方程  得到的方程是$v_i^TK^{-T} K^{-1} v_j = 0$  没有外参矩阵的原因是消掉了  这个方程中只有内参矩阵
对于这组方程 虽然它是非线性的 但它不难解
![](/images/notes/计算机视觉/f2dc40282aa3-06.png)
但需要注意的是至少需要两个有限制的消失点来解出f cx cy
那么如何解出R呢
注意我们对消失点的限制 可以列出$K^{-1} v_i = Re_i$
我们注意到$Re_i = r_i$ 这样R就解出来了
当我们将$||R||^2 = 1$作为限制就不会出现规模不糊不清的问题了
