> 这篇笔记整理一次 POS Tagging 实验中的方法选择与误差分析。重点不是复述作业流程，而是把 HMM、Viterbi、OOV 处理这几个概念放到一个可复现的序列标注场景里理解。

## 任务形式

词性标注可以写成一个标准序列标注问题：

```text
输入: w1, w2, ..., wN
输出: t1, t2, ..., tN
```

其中 `wi` 是句子中的第 `i` 个词，`ti` 是它对应的词性标签。评价时通常按 token 级准确率计算：

```text
Acc = 正确标注的 token 数 / 总 token 数
```

这个任务适合用来比较两类信息：

- 词本身的记忆信息：一个词过去最常见的标签是什么
- 上下文信息：前后标签组合在语言中是否合理

## Majority Voting Baseline

最直接的 baseline 是 majority voting。对训练集中出现过的词，直接预测它最常见的词性：

```text
t_hat(w) = argmax_t count(w, t)
```

如果测试词没有在训练集中出现，就退回到全局最常见标签。

这个 baseline 的优点是简单、稳定，对高频词效果很强；缺点也明显：它无法处理同一个词在不同上下文中的词性变化。例如一个词既可能作名词，也可能作动词时，majority voting 只能给出一个固定答案。

## HMM 的建模假设

隐马尔可夫模型把词性序列看成隐藏状态，把词序列看成观测。二元 HMM 的联合概率可以分解为：

```text
P(w1:N, t1:N)
= P(t1) * Π P(ti | t_{i-1}) * Π P(wi | ti)
```

这对应两类概率：

- 转移概率 `P(ti | t_{i-1})`：相邻词性之间如何接续
- 发射概率 `P(wi | ti)`：给定某个词性时生成某个词的概率

HMM 相比 majority voting 的关键增量在于转移概率。它不只是问“这个词最常是什么标签”，还会问“这个标签放在当前上下文里是否自然”。

## Viterbi 解码

给定一句话后，需要找概率最大的标签序列：

```text
argmax_t1:N P(t1:N | w1:N)
```

直接枚举所有标签序列会指数爆炸。Viterbi 的作用是用动态规划保留每个位置、每个候选标签下的最优前缀路径。

如果标签数为 `K`，句长为 `N`，二元 HMM 的解码复杂度是：

```text
O(N * K^2)
```

这里的 `K^2` 来自每一步需要枚举“上一个标签 -> 当前标签”的转移。

## 平滑为什么必要

训练集中没有出现过的转移或发射不能直接给零概率，否则整条路径会被排除。常见做法是加 `k` 平滑：

```text
P(tj | ti) = (count(ti, tj) + k) / (count(ti) + K * k)
```

发射概率也可以做类似处理。平滑的目的不是把所有没见过的事件都说成“很可能”，而是避免模型因为数据稀疏而过早把合理路径置零。

## OOV 词的处理

OOV 是 POS tagging 中很容易拉开差距的部分。一个常见思路是借鉴 TnT tagger，用稀有词的后缀统计来估计未知词的词性倾向。

例如英文中：

- `-ing` 常提示动名词或现在分词
- `-ly` 常提示副词
- `-ed` 常提示过去式或过去分词

但后缀回退并不总是可靠。如果测试数据被字符扰动过，后缀就可能不再携带真实形态信息。此时纯后缀模型会被噪声带偏。

一个稳健做法是把后缀分布和全局标签先验插值：

```text
log P_oov(w | t)
= alpha * log P(suffix(w) | t)
  + (1 - alpha) * log P(t)
```

这样当后缀信号强时，模型仍能利用形态线索；当后缀信号弱或被扰动时，模型可以退回到更稳的全局高频标签策略。

## 实验现象

在这次实验里，HMM 在三个数据集上都超过了 majority voting baseline。主要收益来自两点：

1. 对已见词，HMM 能用上下文修正同形多性的错误。
2. 对 OOV 词，后缀回退和全局先验插值能减少纯规则回退的风险。

最有意思的是扰动英文数据。它的 OOV 率较高，而且字符扰动会破坏正常后缀。纯后缀回退在这种情况下会显著失效；加入全局先验后，OOV 部分至少能退化到接近 baseline 的水平，同时保留 HMM 在已见词上的上下文优势。

## 复习时抓住什么

这类题的核心不是死记一个模型名字，而是能解释每个部件解决了什么问题：

- majority voting：强词表记忆，但没有上下文
- HMM：用转移概率引入标签上下文
- Viterbi：用动态规划找全局最优标签序列
- 平滑：处理稀疏概率和未见事件
- OOV 回退：为未知词提供发射概率估计
- 先验插值：在形态线索不可靠时保持稳健

如果要继续提升，可以考虑三元 HMM、CRF，或者加入大小写、数字、词形等更细的手工特征。更现代的方案通常会使用字符级或子词级神经表示，但在小数据和可解释实验中，HMM 仍然是很好的基线。
