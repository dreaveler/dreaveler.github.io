> 这篇笔记整理一次对抗鲁棒性实验中的攻击、防御和调参观察。重点是把 PGD、C&W、TRADES 等方法放在同一个实验框架里比较。

## 对抗样本问题

对抗样本的典型现象是：输入图像在人眼看来变化很小，但模型预测发生明显改变。可以把它理解成模型在原样本附近的决策边界过于脆弱。

攻击的目标通常是在扰动预算内最大化分类错误：

```text
maximize   CE(model(x + delta), y)
subject to ||delta|| <= epsilon
```

防御的目标则不是简单提高自然准确率，而是在自然准确率和鲁棒准确率之间找到更好的折中。

## PGD Attack

PGD 可以看作多步 FGSM。每一步沿着让交叉熵损失变大的方向更新扰动：

```text
delta <- delta + alpha * sign(grad_delta CE(model(x + delta), y))
delta <- clip(delta, -epsilon, epsilon)
x_adv <- clip(x + delta, 0, 1)
```

这里有两个约束：

- `delta` 不能超过扰动预算
- `x + delta` 必须仍是合法图像

PGD 常加入 random start，也就是先在扰动球内随机初始化 `delta`。这样可以减少固定起点带来的攻击不足。多次 restart 时，可以保留交叉熵损失最大的一组扰动，作为更强的 untargeted attack 结果。

## C&W Attack

C&W 攻击不是直接做符号梯度步，而是把对抗样本当成一个连续优化变量来求。为了保证图像始终在合法范围内，常用 tanh 重参数化：

```text
adv = 0.5 * (tanh(w) + 1)
```

untargeted C&W 的目标函数可以写成：

```text
loss = ||adv - x||_2^2 + c * f(adv)
f(adv) = max(logit_y - max_{i != y} logit_i, -kappa)
```

第一项控制对抗样本离原图不要太远，第二项推动真实类别 logit 低于其他类别 logit。`kappa` 控制希望跨过分类边界的 margin。

PGD 更像是在给定范数球内直接找高损失方向；C&W 更强调连续优化和扰动距离控制。

## 防御：PGD-AT、TRADES 与 MART

对抗训练的基本思想是在训练时加入对抗样本，让模型学习更平滑、更稳定的局部决策边界。

PGD-AT 的目标直接使用对抗样本上的交叉熵：

```text
min CE(model(x_adv), y)
```

TRADES 则把自然样本分类和局部平滑性分开：

```text
loss = CE(model(x), y)
       + beta * KL(model(x_adv), model(x))
```

这里 `beta` 控制自然准确率和鲁棒性之间的权衡。它的直觉是：自然样本要分对，同时对抗样本附近的预测分布不要剧烈变化。

MART 更关注被模型错分或低置信度分类的样本，通过调整损失让训练更重视这些困难样本。

## 短训练中的调参观察

在很短的训练预算下，鲁棒训练尤其容易出现自然准确率和鲁棒准确率互相拉扯的问题。一次有效的组合是：

- 使用 random start PGD 生成训练用对抗样本
- 用 TRADES 保持自然分类和局部平滑之间的平衡
- 使用 batch-level cosine learning rate，让短训练也能完成学习率退火
- 加入适度 label smoothing，降低模型过度自信
- 在 batch size 上做小范围搜索，观察 clean accuracy 和 robust accuracy 的综合指标

这类实验里，“鲁棒准确率最高”不一定等于最终分数最高。如果评分同时包含 clean accuracy 和 robust accuracy，TRADES 往往能通过更好的自然准确率拿到更高综合分。

## 如何读实验结果

攻击结果首先要看是否真的改变了模型预测；如果对抗图像视觉上仍接近原图，但预测类别已经改变，就说明攻击成功。

防御结果要同时看：

- clean accuracy：自然样本上是否仍能分类正确
- robust accuracy：对抗样本上是否还能保持正确
- score：具体任务定义的综合指标

PGD-AT 和 MART 通常能提高鲁棒性，但可能牺牲较多自然准确率。TRADES 的优势是提供一个显式权衡项，更适合在短训练预算下寻找综合表现。

## 复习时抓住什么

这组方法可以按目标函数来区分：

- PGD：在扰动预算内最大化交叉熵
- C&W：最小化扰动距离，同时推动分类 margin 跨界
- PGD-AT：直接在 PGD 对抗样本上训练
- TRADES：自然 CE 加预测分布平滑约束
- MART：更重视错误样本和困难样本

理解这些方法时，最好一直问同一个问题：它在优化什么，约束是什么，牺牲了哪一侧指标，换来了哪一侧收益。
