> 这篇笔记整理纯交换经济中的一般均衡。重点是把 Edgeworth 方框、帕累托改进、契约曲线、竞争性均衡、福利经济学两个基本定理和 Walras 法则连成一条线：价格不是只在单个市场里起作用，而是同时协调多个市场的供需。

## 从局部均衡到一般均衡

前面的消费者理论、厂商理论和垄断模型大多是在一个市场里分析价格、需求、供给和剩余，这叫局部均衡。局部均衡适合回答“某个商品的价格如何变化”，但它暂时忽略了市场之间的联动。

一般均衡关心的是：多个市场同时存在时，一个市场的需求、供给和价格如何影响另一个市场，最后是否存在一组价格让所有市场同时出清。

最简单的模型是纯交换经济：

- 只有两个消费者 `A` 和 `B`。
- 只有两种商品 `1` 和 `2`。
- 没有生产，社会总资源来自两个人的初始禀赋。
- 每个人在给定价格和禀赋下选择最喜欢的消费组合。

这个模型虽然简单，但已经能说明一般均衡的核心问题：是否交易、交易后的价格是什么、交易结果是否有效率。

## Edgeworth 方框：把所有可行分配画在一张图里

设社会中商品 `1` 的总量为 `w_1`，商品 `2` 的总量为 `w_2`。Edgeworth 方框的宽表示 `w_1`，高表示 `w_2`。

消费者 `A` 的消费从左下角原点 `O_A` 衡量，消费者 `B` 的消费从右上角原点 `O_B` 反向衡量。方框中的任意一点都代表一种可行分配：

```text
x_A1 + x_B1 = w_1
x_A2 + x_B2 = w_2
```

也就是说，只要给定 `A` 的消费，`B` 的消费就由剩余资源自动确定。初始禀赋点表示两个人在交易前分别拥有的商品组合。

Edgeworth 方框的好处是，它把“可行性”和“两个人的偏好”放在同一张图里。`A` 的无差异曲线从 `O_A` 看，`B` 的无差异曲线从 `O_B` 看。由于两个原点方向相反，图上同一个点可以同时读出两个人的消费和福利水平。

## 帕累托改进：为什么会发生交易

如果从初始禀赋出发，存在一种重新分配让至少一个人变好，并且没有人变差，这种分配就是帕累托改进。

在纯交换经济中，交易的基本直觉是：

- 如果存在帕累托改进空间，当前分配还不是稳定结果。
- 因为每个人都可以拒绝交易，所以自愿交易的结果必须让参与者不比原来差。
- 只要两个人的边际替代率不同，就通常存在互利交换的空间。

在 Edgeworth 方框里，初始禀赋点附近两个人都更偏好的区域就是互利交易区域。随着交易进行，这个区域会缩小；当再也找不到同时改善双方福利的重新分配时，就到达帕累托最优。

## 契约曲线：所有有效率分配的集合

一个分配是帕累托最优，意思是不存在进一步的帕累托改进。图形上，如果两个人的偏好满足通常的凸性和单调性，内点处的帕累托最优分配满足两条无差异曲线相切。

用边际替代率表示，就是：

```text
MRS_A = MRS_B
```

直觉是，如果 `A` 愿意用更多商品 `2` 换一单位商品 `1`，而 `B` 愿意用较少商品 `2` 放弃一单位商品 `1`，那么两人之间仍然可以通过交换变得更好。只有当双方的主观交换率一致时，互利交换才耗尽。

所有帕累托最优分配组成契约曲线。契约曲线不是一个唯一点，而是一组有效率分配。效率只说明“没有浪费互利交换机会”，不说明“分配是否公平”。

## 竞争性均衡：价格如何选出契约曲线上的一个点

在完全竞争市场中，每个消费者都是价格接受者。给定价格 `p_1, p_2` 和自己的初始禀赋，每个人先把禀赋按市场价格折成预算，再选择预算线上的最优消费。

消费者 `A` 的预算约束可以写成：

```text
p_1 x_A1 + p_2 x_A2 = p_1 w_A1 + p_2 w_A2
```

消费者 `B` 也有对应的预算约束。预算线的斜率是：

```text
-p_1 / p_2
```

因为禀赋本身一定买得起，预算线经过初始禀赋点。价格变化时，预算线会围绕禀赋点旋转。

竞争性均衡要求两件事同时成立：

1. 给定价格，每个消费者都在自己的预算约束下最大化效用。
2. 每个商品市场都出清：

```text
x_A1 + x_B1 = w_1
x_A2 + x_B2 = w_2
```

如果某个价格下商品 `1` 超额供给、商品 `2` 超额需求，那么 `p_1` 会下降、`p_2` 会上升，预算线相应旋转。均衡价格就是让两个人的最优选择刚好合计等于总禀赋的相对价格。

## 福利经济学第一定理

福利经济学第一定理说明，在适当条件下，完全竞争市场的均衡分配是帕累托最优的。

常见条件包括：

- 消费者偏好良性，例如单调、凸、连续。
- 没有外部性。
- 信息充分。
- 每个参与者都是价格接受者。

定理的直觉是：价格把稀缺性压缩成一个共同的交换率。每个消费者都按同一组相对价格优化，如果市场又出清，那么没有人还能在不损害别人的情况下通过重新分配变得更好。

这不是说竞争性均衡一定公平，也不是说现实市场一定满足这些条件。它说明的是：在理想竞争条件下，市场价格机制可以把分散的个人选择协调成有效率的资源配置。

## 福利经济学第二定理

第一定理从“竞争性均衡”推出“帕累托最优”。第二定理反过来说明，在适当条件下，任意帕累托最优分配都可以通过先调整初始禀赋、再让竞争性市场运行来实现。

可以概括为：

```text
先再分配禀赋 -> 再让市场交易 -> 实现目标帕累托最优点
```

这个定理的重要含义是：效率和公平在理论上可以分开处理。政府若想改变最终分配，不一定要扭曲价格或交易过程；也可以先通过一次性禀赋再分配改变起点，再让市场机制实现有效率的结果。

当然，现实中再分配本身可能有信息、激励和行政成本，所以第二定理更像一个基准框架：它提醒我们，市场机制负责效率，初始权利和禀赋分配决定均衡落在契约曲线上的哪一点。

## Walras 法则：为什么一个市场出清会牵动另一个市场

Walras 法则说，在纯交换经济中，只要消费者花光预算，所有市场超额需求的价值和恒等于零。

设商品 `j` 的总超额需求为：

```text
z_j(p) = x_Aj(p) + x_Bj(p) - w_j
```

Walras 法则写成：

```text
p_1 z_1(p) + p_2 z_2(p) = 0
```

这不是均衡条件，而是由预算约束推出的恒等式。每个消费者都把自己的禀赋价值全部用于购买商品，所以整个经济中“多想买的商品价值”和“少想买的商品价值”必须抵消。

在两商品经济里，它有两个直接推论：

- 如果一个市场已经出清，另一个市场也必须出清。
- 如果商品 `1` 超额供给，那么商品 `2` 必然超额需求；反过来也一样。

这解释了一般均衡为什么不能只看一个市场：一个市场的失衡会以相反方向反映到其他市场。

## 复习抓手

这一讲可以按六个关键词复习：

- Edgeworth 方框：方框内每一点都是一种可行分配。
- 帕累托改进：存在互利交易空间时，当前分配还不是效率终点。
- 契约曲线：所有帕累托最优分配的集合，内点通常满足 `MRS_A = MRS_B`。
- 竞争性均衡：给定价格的个人最优选择加上所有市场出清。
- 第一福利定理：理想竞争均衡是帕累托最优。
- 第二福利定理：适当再分配初始禀赋后，可以用竞争性市场实现任意目标效率点。

最重要的主线是：一般均衡把“价格”理解为全经济范围内的协调机制。价格不仅决定单个市场的供需，也决定每个人预算线如何旋转、交易落到契约曲线上的哪一点，以及市场均衡如何把个体优化转化为整体效率。
