> 北京大学 2026 年春季学期《中级微观经济学》专题整理。内容主要基于 `Lec9 Cost Minimization`、`Lec10a Cost Curves` 与 `Lec10b Firm and Industry Supply` 三份课件，单独梳理从生产函数到成本曲线、再到厂商供给和行业供给的逻辑链。

## 为什么厂商理论要从成本最小化开始

消费者理论的主线是“在预算约束下最大化效用”。厂商理论的主线可以看成对应版本：

1. 给定生产技术和要素价格，先用最便宜的方式生产某个产量
2. 把每个产量对应的最低成本连起来，得到成本函数
3. 给定产品价格，再选择让利润最大的产量
4. 把单个厂商的最优产量加总，就得到市场供给
5. 在长期里，企业可以进入或退出，行业供给还要满足非负利润约束

所以“成本曲线”不是一组孤立图形，而是从技术约束、要素价格和利润最大化一步步推出来的。

## 成本最小化：先固定产量，再选择投入

设厂商用两种投入 `x1, x2` 生产产出 `y`，生产函数为：

```text
y = f(x1, x2)
```

投入价格分别是 `w1, w2`。如果目标是生产恰好 `y` 单位产出，厂商要解的是：

```text
min w1 x1 + w2 x2
s.t. f(x1, x2) >= y
```

这和消费者问题在形式上很像，但方向相反：

- 消费者是在预算线下找最高的无差异曲线
- 厂商是在给定等产量线下找最低的等成本线

等成本线写作：

```text
w1 x1 + w2 x2 = c
```

它的斜率是 `-w1 / w2`。内点最优时，最低等成本线与给定等产量线相切，因此满足：

```text
MRTS = w1 / w2
```

这里 `MRTS` 是边际技术替代率，表示在保持产量不变时，用一种投入替代另一种投入的技术比率。最优条件的含义是：技术上愿意替代的比例，必须等于市场上的要素价格比。

## 条件要素需求和成本函数

成本最小化问题的解通常记为：

```text
x1(w1, w2, y), x2(w1, w2, y)
```

它们叫做条件要素需求函数，因为它们是在“必须生产 `y` 单位产出”这个条件下，对投入的最优选择。

把最优投入代回成本，就得到成本函数：

```text
c(w1, w2, y) = w1 x1(w1, w2, y) + w2 x2(w1, w2, y)
```

这一步非常关键：成本函数已经不直接写投入，而是告诉我们“生产任意产量 `y` 的最低成本是多少”。后面的平均成本、边际成本、供给曲线都从这里出来。

如果生产技术有规模报酬递增，那么产量扩大时平均成本会下降；如果有规模报酬递减，平均成本会上升；如果是规模报酬不变，平均成本可以保持不变。直觉上，规模报酬描述的是“把投入一起放大后，产出放大的幅度”。

## 固定成本、可变成本与平均成本

短期里至少有一种投入固定，因此总成本可以拆成：

```text
c(y) = F + cv(y)
```

其中：

- `F` 是固定成本，和产量无关
- `cv(y)` 是可变成本，随着产量变化

对应的平均成本有三条：

```text
AFC(y) = F / y
AVC(y) = cv(y) / y
AC(y) = c(y) / y = AFC(y) + AVC(y)
```

平均固定成本 `AFC(y)` 随产量增加而下降，并趋近于 0。平均可变成本 `AVC(y)` 通常会因为边际报酬递减而在某个阶段后上升。平均总成本 `AC(y)` 同时受到二者影响：早期固定成本摊薄会拉低它，后期可变成本上升会推高它，所以常见形状是 U 形。

## 边际成本为什么穿过平均成本最低点

边际成本是多生产一单位产出增加的成本：

```text
MC(y) = dc(y) / dy
```

它和平均成本的关系可以用一个简单规则记住：

- 如果 `MC(y) < AC(y)`，新增这一单位比当前平均成本低，会把平均成本拉下来
- 如果 `MC(y) > AC(y)`，新增这一单位比当前平均成本高，会把平均成本推上去
- 如果 `MC(y) = AC(y)`，平均成本正处在极值点

因此，边际成本曲线从下往上穿过平均成本曲线的最低点。同理，`MC` 也会从下往上穿过 `AVC` 的最低点。

这不是图形记忆题，而是“边际量与平均量”的通用关系。只要边际值低于平均值，平均值就下降；只要边际值高于平均值，平均值就上升。

## 短期供给：价格等于边际成本，但要看停产点

厂商利润为：

```text
pi(y) = p y - c(y)
```

利润最大化的一阶条件是：

```text
p = MC(y)
```

但这只是内点候选解。短期里固定成本已经无法避免，所以厂商是否生产，要比较生产和停产：

- 如果生产，利润是 `p y - F - cv(y)`
- 如果停产，利润是 `-F`

生产至少要覆盖可变成本，即：

```text
p y >= cv(y)
```

等价于：

```text
p >= AVC(y)
```

所以短期供给曲线是边际成本曲线在 `AVC` 最低点以上的部分。价格低于最低平均可变成本时，厂商会停产；价格高于停产点时，厂商选择满足 `p = MC(y)` 的产量。

## 长期供给：企业进入退出会压到零利润

长期里所有投入都可变，企业也可以进入或退出。此时固定成本不再是不可避免的沉没项，厂商要能覆盖全部成本。

长期中单个厂商愿意生产的最低价格是：

```text
p* = min AC(y)
```

如果市场价格低于 `min AC`，任何产量都会带来负利润，企业会退出；如果价格高于 `min AC`，现有企业有正经济利润，新企业会进入。

这解释了 `Lec10b` 中行业供给的核心图像：

- 市场需求给出某个价格
- 给定企业数量时，市场供给是各厂商供给的水平加总
- 如果价格高于 `min AC`，正利润吸引进入，市场供给右移，价格下降
- 如果价格低于 `min AC`，企业退出，市场供给左移，价格上升
- 长期均衡企业数量，是市场能够容纳且价格不低于 `min AC` 的最大数量

在同质企业、自由进入退出的标准模型中，长期均衡通常会把价格压到最低平均成本附近，单个企业获得零经济利润。

## 短期和长期的区别

最容易混淆的是“短期停产”和“长期退出”。

短期里固定成本已经付出，即使停产也要承担，所以只要价格能覆盖平均可变成本，厂商就可能继续生产以减少亏损。长期里所有成本都要重新考虑，如果价格无法覆盖平均总成本，企业不会留在行业中。

可以压缩成两条规则：

```text
短期生产条件：p >= min AVC
长期留在行业：p >= min AC
```

## 复习时最该连起来的几句话

厂商理论的完整链条可以这样背：

1. 等产量线给出技术约束，等成本线给出要素价格约束
2. 成本最小化的切线条件是 `MRTS = w1 / w2`
3. 条件要素需求代回去，得到成本函数 `c(y)`
4. 平均成本是总量除以产量，边际成本是总成本的导数
5. `MC` 从下往上穿过 `AVC` 和 `AC` 的最低点
6. 短期供给是 `MC` 在最低 `AVC` 以上的部分
7. 长期供给还要满足价格不低于最低 `AC`
8. 自由进入退出会让长期经济利润趋向零

如果这些句子能一口气接上，成本曲线、厂商供给和行业供给就不再是三块分散内容。
