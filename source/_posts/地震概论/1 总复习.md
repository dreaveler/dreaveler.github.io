<span color="orange">纵波与横波</span>
- <span color="orange">纵波：振动方向与能量传播方向一致  </span>$v = \sqrt{E/\rho}$
- <span color="orange">横波：振动方向与能量传播方向垂直  </span>$v = \sqrt{\mu/\rho}$
- <span color="orange">纵波速度比横波速度大 一般为 </span>$\sqrt{3}$<span color="orange"> 倍</span>
- <span color="orange">因此纵波先到达 称为P波   横波称为S波</span>
$\mu$为剪切模量
体波：在三维空间中传播的波 包括P波 S波
<span color="red">P波与S波的差异：</span>
- <span color="red">P波比S波快</span>
- <span color="red">质点震动方向垂直</span>
- <span color="red">三分量地震图上 P波的垂直分量相对较强 S波的水平分量相对较强</span>
- <span color="red">S波的低频成分比P波丰富</span>
- <span color="red">天然震源的震源破裂通常剪切破裂和剪切错动为主 震源向外辐射的S波的能量比P波的强</span>
- <span color="red">P波通过时 质元无转动运动 有体积变化 P波是一种无旋波 S波通过时 质元有转动 无体积变化 S波是一种无散的等容波</span>
S波可以分解为SV波 SH波
面波的能量是在表面传播的
若介质是分层的  首波是最先到达的波 可能路径比直接到达长
震源到地表的投影为震中 震中距是震中到观测台间的距离  可以用km或度来表示
射线理论 短波/高频近似条件 波震面法线的轨迹 由费尔马原理推导得到

五 地震机理
断层：沿破裂面两侧岩块发生显著相对位移的构造
各大板块之间都是断层
岩石变形：冷的岩石容易发生脆性断裂 即容易地震
地震发生在断层上 断层滑动开始的位置叫做震源
重要术语：
- 上盘/下盘
- <span color="purple">走向：断层与水平面的交线是一条直线 它有两个方向 如果上盘在右边 就是断层走向的方向  而其度数是由正北方向作为0°  顺时针旋转到这个角度</span>
- 倾角：断层面与水平面/地球表面的夹角
- 滑移：上盘相对于下盘的方向为滑移方向


断层类型：根据滑移方向进行分类
- 正断层：沿断层上盘向下  下盘向上
- 逆断层：沿断层上盘向上 下盘向下
- 走滑断层：水平方向的侧滑   分为左旋右旋 ：站在一边上面看另外一边 它向哪边就是什么旋
- 斜滑断层：断层倾斜的侧滑
