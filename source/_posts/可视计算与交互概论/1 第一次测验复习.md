
**Bresenham's Line Algorithm(布雷森汉姆直线算法)**
核心思想是累积误差
让我们先假设直线的斜率0 \< k \< 1  这样x轴移动1pixel   y轴最多加1pixel
对于y与y\*(绘制位置)  有一个err 当这个err≥0.5时  y应该+1  err\<0.5时 y应该不变
而我们就可以累加这个err
同时我们可以将err放大到2倍 这样可以保证是一个整型  我们就可以修正每次的判断条件为 $2 \times err \times \Delta x - \Delta x >= 0$
又因为右边是0  所以我们可以乘上对应的倍数   即可以用初始的y0 y1  x0  x1来表示
我们就得到了决策变量  F = 2 \* dy \* dx - dx
当其≥0时  我们应该令y + 1   原先的err -1  对应到F里应该就是 + 2 \* (dy - dx)   这里是+2\*dy是累积误差   -2dx对应的是-1
else  y不变  累积误差  +2\*dy
```python
def draw_line(x0: int, y0: int, x1: int, y1: int):
  y = y0
  dx, dy = 2 * (x1 - x0), 2 * (y1 - y0)
  dydx, F = dy - dx, dy - dx // 2
  for x in range(x0, x1 + 1):
    draw_pixel(x, y)
    if F < 0:
      F += dy
    else:
      y += 1
      F += dydx
```
那么只有0\<k\<1这个条件没讨论了  对于8个区域进行分类讨论即可


布雷森汉姆圆形算法
与直线算法相仿  需要注意的是圆有八分对称性 即得到了一个点的位置 可以同时得到另外7个点的位置  分别是 (y,x)  (-y,x)  (-x,y)  (x,-y)  (-x,-y)  (y,-x) (-y,-x)  这也就意味着我们只需要计算0-45°这个区间的圆的像素位置就可以画出整个圆
我们以(0,R)开始  分析x+1时 y是否需要-1  那么我们可以每次分析(x+1,y-0.5)是否在圆内  如果在圆内说明y离圆的边界更近 即不需要减去  在圆外说明y-1离圆的边界更近
重新完成决策变量$F =  (x_i+1)² + (y_i - 0.5)² - R²$
1. **如果 ****`p_i < 0`**，选中点 `H(x_i+1, y_i)`。
	- 那么下一个中点 `M_next` 的坐标是 `(x_i+2, y_i - 0.5)`。
	- 决策变量的更新为：
		`p_{i+1} = F(x_i+2, y_i - 0.5)`
		`= (x_i+2)² + (y_i - 0.5)² - R²`
		`= (x_i+1)² + (y_i - 0.5)² - R² + 2x_i + 3`
		`= p_i + 2x_i + 3`
2. **如果 ****`p_i >= 0`**，选中点 `D(x_i+1, y_i-1)`。
	- 那么下一个中点 `M_next` 的坐标是 `(x_i+2, y_i - 1.5)`。
	- 决策变量的更新为：
		`p_{i+1} = F(x_i+2, y_i - 1.5)`
		`= (x_i+2)² + (y_i - 1.5)² - R²`
		`= (x_i+1)² + (y_i - 0.5)² - R² + 2x_i + 3 + (-2y_i + 2)`
		`= p_i + 2x_i - 2y_i + 5`
	为了消除浮点数  我们可以把整体\*4
```c++
void draw_circle(int radius) {
	int x = 0, y = radius;
	int d = 3-2*radius;
	while (y>x) {
		if (d<0) /* select East point next */
			d += 4*x + 6;
		else { /* select South-East point next */
			d += 4*(x-y) + 10;
			y--;
		}
		x++;
		draw_8_pts(x,y); /* draws point in each octant */
	}
}
```

**扫描线算法**
这个算法的核心就是一行一行绘制pixel  直到左端点与右端点时中间的pixel不需要进行判断  而两个端点的pixel实际上就是DDA算法
```python
def draw_triangle(T):
  for each edge pair:
  initialize x_L, x_R
  compute dx_L/dy_L, dx_R/dy_R
  for each scanline at y:
    for x in range(x_L, x_R + 1):
    draw(x, y)
    x_L += dx_L/dy_L
    x_R += dx_R/dy_R
```

**贝塞尔曲线**
定义非常简单 但具有良好的几何性质
对于三个控制点 P0  P1  P2   通过两次线性插值得到Q0 Q1  再进行一次线性插值可以得到点S  这3次线性插值共用一个参数t  而将t从0到1遍历  就可以得到S所连接成的贝塞尔曲线
更高阶同理
需要注意的是  对于高阶贝塞尔曲线 由于改变一个控制点会同时改变整体曲线状态  会使曲线变得不可控  因此一般使用多个低价贝塞尔曲线连接起来 一般是用3阶贝塞尔曲线 即有4个控制点的  很容易就可以保证G0 G1连续
此外 通过这种方式我们可以得到贝塞尔曲线的数学表达式   不同控制点的系数正是二项式展开后的各项  我们也可以得到贝塞尔曲线的基函数是：$C_n^k(1-t)^kt^{n-k}$
这样我们可以使用矩阵来对贝塞尔曲线进行表示

**样条曲线**
插值样条与逼近样条
插值样条是经过曲线控制点的  逼近并不
插值没有凸包性质  而逼近有


插值样条
常见的选择是三次样条
对于每一段Pi  Pi+1之间的样条  假设$x = at^3 + bt^2 + ct + d$  $y = et^3 + ft^2 + gt + h$
由于x与y是独立的  下面只考虑x
对于点列\{P0,…,Pm)  我们需要列出m个方程  也就有4m个参数
而由于每个曲线必通过左右端点  我们就可以列出2m个方程
我们可以要求每个点左右连续  也就是 $f_i^'(1)  = f_{i+1}^'(0)$  这样的方程我们可以列出2m-2个
还有两个可以自行指定  如果指定左右端点的二阶导是0 这样称为自然边界条件  这样我们就获得了4m个方程 可以解得需要的参数了
会发现这样的样条是C2的  但是改变一个点就需要求解整个曲线  这个特性称之为非局部性


三次hermite样条  不要求二阶导连续  换来的是可以指定每个点的导数
对于每一段曲线 我们可以列出方程组为：  $f_i(0) = x_i$    $f_i(1) = x_{i+1}$   $f^'_i(0) = pi$  $f^'_i(1) = p_{i+1}$

**B样条**

显然p越大一个控制点就会影响更多的区域  局部性也就越差

**图像分割与抠图**
大部分图都有前景与背景  而图像分割一般指前景与背景不混合在一起的情况  抠图指前景与背景混合的情况  图像分割是抠图的一个特例
对于每个点 我们可以列出方程  $C = \alpha F + (1-\alpha)B$  又可以RGB不同通道分别列出
分别列出后显然我们有三个方程  七个未知量  这时不可解的
**蓝绿幕抠图**
其原理很简单 就是背景以蓝色/绿色出现 ，我们也假设前景没有对应颜色 这样另外两个通道的背景以及对应通道的前景就没有未知量了，而我们也知道背景蓝绿幕的颜色  对应通道的未知量也变为了已知量   这样就是三个未知量三个方程  是可解的
