> 这篇笔记整理中级微观经济学里“垄断与价格歧视”的主线。重点不是背图，而是理解三件事：垄断厂商为什么按 `MR = MC` 定产量，为什么统一定价会带来无谓损失，以及价格歧视和两部定价怎样改变剩余分配与效率。

## 从完全竞争到垄断

完全竞争厂商是价格接受者，面对的价格不随自身产量变化；垄断厂商是单一卖者，面对整条向下倾斜的市场需求曲线。因此垄断厂商可以通过减少产量提高市场价格，也必须在“多卖一单位”和“降价影响所有单位收入”之间权衡。

设反需求函数为

```text
p = p(y)
```

总收益为

```text
R(y) = p(y)y
```

边际收益是

```text
MR(y) = R'(y) = p(y) + y p'(y)
```

因为需求曲线向下倾斜，`p'(y) < 0`，所以只要 `y > 0`，就有

```text
MR(y) < p(y)
```

这就是垄断定价和完全竞争定价的核心差别：垄断厂商多卖一单位时，不只获得新增那一单位的价格，还要承受为了卖出更多而降低价格的收入损失。

## 利润最大化：MR = MC

垄断厂商选择产量 `y` 最大化利润：

```text
pi(y) = p(y)y - c(y)
```

一阶条件是

```text
MR(y*) = MC(y*)
```

若需求是线性的 `p(y) = a - by`，则

```text
MR(y) = a - 2by
```

边际收益曲线比需求曲线下降得更快。图形上，垄断产量由 `MR` 和 `MC` 的交点决定，价格再从需求曲线上读出。由于 `MR < p`，垄断均衡通常满足

```text
p(y*) > MC(y*)
```

这意味着消费者对边际单位的支付意愿高于生产该单位的边际成本，但交易没有发生。

## 弹性、加成定价与垄断价格

令需求价格弹性为

```text
epsilon = (dy/dp) * (p/y)
```

对向下倾斜需求，`epsilon < 0`。边际收益可以写成

```text
MR = p(1 + 1/epsilon)
```

利润最大化要求 `MR = MC`，于是

```text
p = MC / (1 + 1/epsilon)
```

这个式子说明，垄断厂商只会在需求富有弹性的区间生产，因为若 `|epsilon| <= 1`，边际收益非正，不可能等于正的边际成本。

它也给出“成本加成定价”的直觉：需求越缺乏弹性，消费者对价格越不敏感，垄断厂商能收取的加成越高。例如当 `epsilon = -4` 时，加成较小；当 `epsilon = -2` 时，加成更大。

## 利润税与从量税

不同税种对垄断行为的影响不同。

利润税按利润比例征收。若税率为 `t`，税后利润是

```text
(1 - t) pi(y)
```

只要 `t` 不随产量改变，最大化税后利润和最大化税前利润选择的是同一个 `y*`。所以利润税不改变垄断厂商的产量、价格和要素需求，常被称为中性税。

从量税按每单位产量征收，等价于把边际成本从 `MC(y)` 提高到

```text
MC(y) + t
```

新的利润最大化条件变成

```text
MR(y_t) = MC(y_t) + t
```

因此从量税会降低产量、提高消费者价格，并减少要素需求，是扭曲性的。若边际成本不变且需求弹性保持不变，消费者价格上涨幅度甚至可能超过税额本身；但在线性需求下，弹性随产量变化，不能直接套用常弹性结论。

## 为什么统一垄断定价无效率

帕累托有效要求不存在一种重新安排，能让某些人变好而不让任何人变坏。对单一产品市场，效率产量满足

```text
p(y_e) = MC(y_e)
```

因为只要消费者愿付价格高于边际成本，生产这一单位就能创造正的总剩余。

垄断厂商按 `MR = MC` 选择 `y*`。由于 `MR < p`，通常有

```text
y* < y_e
p(y*) > p(y_e)
```

垄断厂商通过限制产量提高价格，获得更高利润，但 `y*` 到 `y_e` 之间本来可以产生的交易没有发生，形成无谓损失。这个损失不是简单的消费者剩余转移，而是社会总剩余的真实减少。

## 自然垄断与监管难题

自然垄断来自规模经济：一个厂商服务整个市场的平均成本低于多个厂商分开服务。典型例子是供水、供电等基础设施行业。

自然垄断同样会按 `MR = MC` 选择低于效率水平的产量，造成无谓损失。直觉上，监管者可以要求它按边际成本定价，让 `p = MC`。问题是，在强规模经济下，效率产量处可能满足

```text
p(y_e) < ATC(y_e)
```

此时边际成本定价会让厂商亏损并退出市场。监管自然垄断的难点就在于：既要提高产量、降低无谓损失，又要让厂商至少覆盖成本。

## 三类价格歧视

价格歧视指厂商对不同单位、不同购买数量或不同消费者群体收取不同价格。它要求厂商具有一定市场势力，并且能阻止低价购买者转卖给高价购买者。

一、一级价格歧视：

厂商能识别每个消费者对每一单位产品的最高支付意愿，并逐单位收费。结果是产量达到效率水平，因为只要 `p >= MC` 的交易都能发生；但消费者剩余全部被厂商提取，买方剩余为零。一级价格歧视可以帕累托有效，但剩余分配极端偏向厂商。

二、二级价格歧视：

厂商不能直接识别消费者类型，但可以设计所有人都可选择的“数量-价格套餐”，让不同类型消费者自我选择。例如批量折扣、会员套餐、买二送一。核心约束是激励相容：高需求消费者必须愿意选择为他设计的套餐，而不是伪装成低需求消费者。

三、三级价格歧视：

厂商能把消费者分成可识别群体，对不同群体设置不同统一价格。例如学生票和普通票。若两个市场分别有反需求 `p_1(y_1)`、`p_2(y_2)`，利润最大化要求

```text
MR_1(y_1*) = MR_2(y_2*) = MC(y_1* + y_2*)
```

价格更高的市场通常是需求更缺乏弹性的市场。直觉是，价格不敏感的群体更容易承受高加成。

## 二部定价

二部定价由固定费用和单位价格组成：

```text
总支付 = p_1 + p_2 x
```

其中 `p_1` 是入场费或会员费，`p_2` 是每单位使用价格。若只有一种消费者类型，厂商最优做法通常是：

```text
p_2 = MC
p_1 = 消费者在 p_2 下获得的消费者剩余
```

单位价格等于边际成本时，消费者购买到效率产量，总剩余最大；固定费用再把消费者剩余转移给厂商。于是二部定价可以同时实现效率产量和高利润。和统一垄断定价相比，它减少了无谓损失，但改变了剩余分配。

现实中的会员制商店、游乐园门票加项目收费、平台会员加按量服务，都可以用二部定价理解。若消费者类型不同，最优二部定价会更复杂，因为统一入场费可能排除低需求消费者。

## 复习抓手

这一章可以用五个判断串起来：

- 垄断厂商面对向下倾斜需求，因此 `MR < p`，利润最大化是 `MR = MC`，不是 `p = MC`。
- 统一垄断定价会让 `p > MC`、`y* < y_e`，所以产生无谓损失。
- 利润税不改变最优产量和价格；从量税提高边际成本，通常降低产量并提高价格。
- 一级价格歧视和二部定价都可能恢复效率产量，但会把更多剩余转给厂商。
- 三级价格歧视的价格差异来自弹性差异：需求越缺乏弹性，垄断加成越高。

最重要的主线是：垄断问题不是“厂商能不能定高价”这么简单，而是市场势力如何改变产量、价格、剩余分配与效率。统一定价牺牲效率，复杂定价可能恢复效率，但通常伴随更强的剩余提取。
