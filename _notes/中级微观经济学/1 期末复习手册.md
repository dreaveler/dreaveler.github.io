## 1. 预算约束

### 核心概念

| 术语 | 定义 |
| --- | --- |
| 预算约束线 | 恰好花光收入时可购买的全部商品组合 |
| 预算集 | 所有可负担组合的集合 |
| 机会成本 | 多获得 1 单位商品 1 需要放弃的商品 2 数量 |

两商品预算约束：

```text
p1x1 + p2x2 = m
斜率 = -p1/p2
横截距 = m/p1
纵截距 = m/p2
```

比较静态：

- 收入上升，预算线平行外移
- `p1` 下降，预算线绕纵截距向外旋转
- 两价格同比例上升，可等价理解为实际收入下降

特殊情形：

- 从量税：等价于价格上升
- 从价税：预算集整体缩小
- 食品券：预算线可能出现折点
- 数量折扣：预算约束分段线性
- 计价物标准化：可把某一商品价格记为 1

## 2. 偏好与效用

### 理性偏好公理

- 完备性
- 自反性
- 传递性

### 无差异曲线与边际替代率

```text
MRS = dx2/dx1 | U=const = -MU1/MU2
```

- 无差异曲线不能相交
- 两种商品都是好商品时，离原点越远效用越高
- 凸形无差异曲线对应边际替代率递减

常见偏好类型：

| 类型 | 形式 | 特征 |
| --- | --- | --- |
| 完全替代品 | `U = ax1 + bx2` | 无差异曲线为直线，`MRS` 为常数 |
| 完全互补品 | `U = min(ax1, bx2)` | L 形无差异曲线 |
| 拟线性偏好 | `U = f(x1) + x2` | 收入效应常较弱 |
| 柯布道格拉斯 | `U = x1^a x2^b` | 平滑、严格凸、常见于标准题 |

要点：

- 效用是序数概念，只有排序有意义
- 严格单增变换不改变偏好
- 良态偏好通常意味着严格单调且严格凸

## 3. 消费者最优选择

### 内点解

最优消费组合 `(x1*, x2*)` 满足：

```text
预算耗尽：p1x1* + p2x2* = m
切线条件：MRS = -p1/p2
```

### 典型需求函数

柯布道格拉斯 `U = x1^a x2^b`：

- `x1* = am / [(a+b)p1]`
- `x2* = bm / [(a+b)p2]`

### 角点解

- 完全替代品：常出现“全买其中一种”
- 完全互补品：最优点落在折点上

对完全替代品要优先比较单位效用价格比；对完全互补品要优先写出固定搭配关系。

## 4. 需求分析

重点区分：

- 价格提供曲线
- 普通需求曲线
- 收入提供曲线
- 恩格尔曲线

商品分类：

| 类型 | 含义 |
| --- | --- |
| 正常品 | 收入上升，需求上升 |
| 低档品 | 收入上升，需求下降 |
| 吉芬商品 | 价格上升，需求反而上升 |

交叉价格效应：

- 总替代品：`∂x1*/∂p2 > 0`
- 总互补品：`∂x1*/∂p2 < 0`

复习时应把“需求曲线”“收入变化”“价格变化”“替代/互补关系”分开判断，不要混在一起。

## 5. 显示偏好

核心思路：通过选择结果反推偏好关系。

- 直接显示偏好（DRP）
- 间接显示偏好（IRP）
- WARP：弱显示偏好公理
- SARP：强显示偏好公理

判断关键：

- 若组合 `y` 在预算下可负担，但消费者选择了 `x`，则 `x` 直接显示偏好于 `y`
- 若出现循环，就可能违反 WARP 或 SARP

数量指数与价格指数也常和显示偏好一起考，用于比较不同状态下的福利高低。

## 6. 斯勒茨基方程

这是整门课最常考的核心部分之一，需要明确区分：

- 总效应
- 替代效应
- 收入效应

标准分解步骤：

1. 先画出价格变化前后的预算线。
2. 构造补偿预算线，把购买力保持在原组合处。
3. 原均衡到补偿均衡是替代效应。
4. 补偿均衡到新均衡是收入效应。

变化率形式：

```text
Δx1/Δp1 = (Δx1/Δp1)|SE + (-x1) · (Δx1/Δm)
```

不同商品类型下：

| 商品类型 | 替代效应 | 收入效应 | 总效应 |
| --- | --- | --- | --- |
| 正常品 | 与价格变化反向 | 与价格变化反向 | 需求法则成立 |
| 低档品 | 与价格变化反向 | 可能同向 | 总效应不确定 |
| 吉芬商品 | 与价格变化反向 | 同向且更强 | 可能逆需求法则 |

## 7. 禀赋与跨期选择

### 禀赋经济

若消费者拥有初始禀赋 `(w1, w2)`，则预算约束为：

```text
p1x1 + p2x2 <= p1w1 + p2w2
```

价格变化时，预算线绕禀赋点旋转，而不是简单平移。

### 劳动供给

- 闲暇可以视作一种商品
- 工资上升同时带来替代效应和收入效应
- 因而劳动供给曲线可能向后弯曲

### 跨期消费

```text
c1 + c2/(1+r) = m1 + m2/(1+r)
```

- 斜率为 `-(1+r)`
- 储蓄者与借款者在利率变化下的福利影响不同

## 8. 消费者剩余

常考内容：

- 保留价格与需求曲线的关系
- 消费者剩余的几何意义
- 价格变化对福利的影响
- CV、EV 与消费者剩余的比较

一般关系：

```text
EV <= ΔCS <= CV
```

若是拟线性效用，则三者会重合。

## 9. 核心公式速查

| 公式名称 | 表达式 |
| --- | --- |
| 预算约束 | `p1x1 + p2x2 = m` |
| 边际替代率 | `MRS = -MU1/MU2` |
| 内点最优 | `MRS = -p1/p2` 且 `p1x1* + p2x2* = m` |
| 柯布道格拉斯需求 | `x1* = am / [(a+b)p1]`，`x2* = bm / [(a+b)p2]` |
| 完全互补品需求 | `ax1* = bx2*` |
| 斯勒茨基方程 | `Δx1/Δp1 = SE + IE` |
| 含禀赋斯勒茨基 | `Δx1/Δp1 = SE + (w1-x1)·(Δx1/Δm)` |
| 跨期预算约束 | `c1 + c2/(1+r) = m1 + m2/(1+r)` |
| WARP | 若 `x` 直显于 `y`，则 `y` 不可直显于 `x` |
