## 1. 预算约束

### 1.1 核心概念

| 术语 | 定义 |
|------|------|
| **预算约束线**（Budget Constraint） | 消费者恰好花光收入时可购买的所有组合的集合 |
| **预算集**（Budget Set） | 所有可负担组合的集合（约束线及其内侧） |
| **机会成本**（Opportunity Cost） | 多获得一单位商品 1 需放弃的商品 2 数量 = p1/p2 |

**两商品预算约束：**
```text
p1x1 + p2x2 = m

斜率 = -p1/p2
横截距 = m/p1
纵截距 = m/p2
```

### 1.2 比较静态

| 变化 | 效果 |
|------|------|
| 收入 m ↑ | 预算线向外**平行移动**（斜率不变） |
| 收入 m ↓ | 预算线向内平行移动 |
| p1 ↓ | 以纵截距为轴**向外旋转**（横截距变大） |
| p1 ↑ | 以纵截距为轴向内旋转 |
| 两价格同比例 ↑ | 等价于收入同比例下降，预算集缩小 |

### 1.3 特殊预算约束

- **从量税（Specific Tax）**：对每单位商品征税 t，等价于价格从 p1 升至 p1 + t，约束线斜率变陡
- **均匀从价税（Ad Valorem Tax）**：税率 t，使价格变为 (1+t)p1。均匀销售税等价于收入下降至 m/(1+t)
- **食品券（Food Stamps）**：实物补贴，扩大预算集但形成折段（kink）；若可在黑市按比例兑现，则等效于部分现金补贴
- **数量折扣/惩罚**：购买超过某阈值后单价改变，预算约束线出现折点
- **计价物（Numeraire）**：将某商品价格标准化为 1，不改变预算集形状

## 2. 偏好与效用

### 2.1 偏好关系的理性公理

| 公理 | 含义 |
|------|------|
| **完备性**（Completeness） | 任意两组合均可比较 |
| **自反性**（Reflexivity） | x ~ x（自身与自身无差异） |
| **传递性**（Transitivity） | x ≻ y 且 y ≻ z ⟹ x ≻ z |

三条公理同时满足，得到**理性偏好（Rational Preferences）**。

### 2.2 无差异曲线

- 无差异曲线：偏好程度相同的所有组合的轨迹
- 两条不同无差异曲线**不可相交**（传递性保证）
- 两种商品都是好商品时，离原点越远效用越高
- **弱偏好集 WP(x)**：所有弱偏好于 x 的组合（含无差异曲线本身）

### 2.3 边际替代率（MRS）

```text
MRS = dx2/dx1 | U=const = -MU1/MU2
```

- MRS 是无差异曲线的斜率（通常为负值）
- 凸形无差异曲线表示 MRS 绝对值沿曲线从左至右**递减**
- 消费者在该点愿意用商品 1 替代商品 2 的比率

### 2.4 偏好的重要性质

| 性质 | 含义 |
|------|------|
| **单调性**（Monotonicity） | 更多总是更好，无差异曲线负斜率 |
| **凸性**（Convexity） | “平均优于极端”，x 与 y 的混合组合弱优于两端 |
| **良态偏好** | 严格单调 + 严格凸，常保证最优解唯一且位于内点 |

### 2.5 特殊偏好类型

| 类型 | 效用函数 | 无差异曲线 | MRS |
|------|----------|------------|-----|
| **完全替代品** | U = ax1 + bx2 | 斜率为常数的直线 | -a/b（常数） |
| **完全互补品** | U = min{ax1, bx2} | L 形，顶点在 ax1 = bx2 射线上 | 0、无穷或未定义 |
| **拟线性** | U = f(x1) + x2 | 垂直方向平移的相同曲线 | 仅依赖 x1 |
| **柯布道格拉斯** | U = x1^a x2^b | 凸形，不与坐标轴相交 | -(ax2)/(bx1) |

### 2.6 效用函数要点

- 效用是**序数**概念，绝对数值无意义，只有排序有意义
- **严格单增变换**后代表相同偏好，MRS 不变
- 连续性 + 理性公理，通常可由连续效用函数表示偏好
- MU1 = ∂U/∂x1，MU2 = ∂U/∂x2，则 **MRS = -MU1/MU2**

## 3. 消费者最优选择

### 3.1 内点解（Interior Solution）

最优消费组合 `(x1*, x2*)` 满足两个条件：

```text
条件 1（预算耗尽）：p1x1* + p2x2* = m
条件 2（切线条件）：MRS = -p1/p2
```

> 直觉：消费者的主观支付意愿等于市场机会成本。

### 3.2 柯布道格拉斯需求（U = x1^a x2^b）

```text
x1* = am / [(a+b)p1]
x2* = bm / [(a+b)p2]
```

- x1 的支出占收入比例恒为 a/(a+b)
- x2 的支出占收入比例恒为 b/(a+b)

### 3.3 角点解（Corner Solution）

**完全替代品** `U = x1 + x2`：

| 条件 | 最优解 |
|------|--------|
| p1 < p2 | x1* = m/p1，x2* = 0 |
| p1 > p2 | x1* = 0，x2* = m/p2 |
| p1 = p2 | 预算线上任意组合均最优 |

**完全互补品** `U = min{ax1, bx2}`：最优点位于折点 `ax1* = bx2*`

```text
x1* = m / (p1 + (b/a)p2)
x2* = (b/a) * x1*
```

## 4. 需求分析

### 4.1 价格提供曲线与需求曲线

- **价格提供曲线（Price Offer Curve）**：其他条件不变，最优组合随某价格变化的轨迹
- **普通需求曲线**：从价格提供曲线投影到价格-数量坐标系得到

| 偏好类型 | 价格提供曲线特征 |
|----------|-----------------|
| 柯布道格拉斯 | x2* 不随 p1 变化，x1 需求曲线近似双曲线 |
| 完全互补品 | x1 和 x2 同比例下降，价格提供曲线更接近垂直 |
| 完全替代品 | 在 p1 = p2 附近突变，呈阶梯状 |

### 4.2 收入效应与恩格尔曲线

- **收入提供曲线（Income Offer Curve）**：固定价格，最优组合随收入变化的轨迹
- **恩格尔曲线（Engel Curve）**：需求量与收入的关系图

| 商品类型 | 定义 | 恩格尔曲线 |
|----------|------|------------|
| **正常品**（Normal） | 收入↑，需求量↑ | 正斜率 |
| **低档品**（Inferior） | 收入↑，需求量↓ | 负斜率 |
| **位似偏好**（Homothetic） | MRS 沿原点射线不变 | 过原点直线 |
| **拟线性偏好** | 某一商品需求不随收入变化 | 对应垂直或线性关系 |

### 4.3 普通商品与吉芬商品

- **普通商品（Ordinary Good）**：p1↑，需求量↓
- **吉芬商品（Giffen Good）**：p1↑，需求量↑

> 价格与需求量同向变动并不自动意味着吉芬商品，必须排除其他因素。

### 4.4 交叉价格效应

- **总替代品（Gross Substitutes）**：∂x1*/∂p2 > 0
- **总互补品（Gross Complements）**：∂x1*/∂p2 < 0
- 完全互补品是总互补品；柯布道格拉斯常见交叉导数为 0

## 5. 显示偏好

### 5.1 核心思想

不直接观察偏好，而是通过预算约束下的选择结果来**反推**偏好信息。

### 5.2 显示偏好关系

```text
直接显示偏好（DRP）：在预算 A 下，y 可负担而 x* 被选择
=> x* 直接显示偏好于 y

间接显示偏好（IRP）：x* 显示偏好于 y，y 又显示偏好于 z
=> x* 间接显示偏好于 z
```

### 5.3 两个公理

**弱显示偏好公理（WARP）**：

> 若 x 直接显示偏好于 y，则 y 不能直接显示偏好于 x。

**强显示偏好公理（SARP）**：

> 若 x 直接或间接显示偏好于 y，则 y 不能直接或间接显示偏好于 x。

- WARP 是理性选择的必要条件
- SARP 更强，用于排除间接循环

### 5.4 指数与福利比较

- 数量指数、价格指数常与显示偏好一起出现
- 核心是利用选择一致性比较不同状态下的福利高低

## 6. 斯勒茨基方程

这是整门课最常考的核心部分之一，需要明确区分：

- **总效应**
- **替代效应**
- **收入效应**

### 6.1 标准分解步骤

1. 画出价格变化前后的预算线。
2. 构造补偿预算线，使原组合刚好可负担。
3. 原均衡到补偿均衡，是替代效应。
4. 补偿均衡到新均衡，是收入效应。

### 6.2 变化率表达

```text
dx1/dp1 = (dx1/dp1)|SE - x1 * (dx1/dm)
```

### 6.3 不同商品类型下的结论

| 商品类型 | 替代效应 | 收入效应 | 总效应 |
|----------|----------|----------|--------|
| 正常品 | 与价格变化反向 | 与价格变化反向 | 需求法则成立 |
| 低档品 | 与价格变化反向 | 可能同向 | 总效应不确定 |
| 吉芬商品 | 与价格变化反向 | 同向且更强 | 可能逆需求法则 |

## 7. 禀赋与跨期选择

### 7.1 禀赋经济

若消费者拥有初始禀赋 `(w1, w2)`，则预算约束为：

```text
p1x1 + p2x2 <= p1w1 + p2w2
```

价格变化时，预算线绕禀赋点旋转，而不是简单平移。

### 7.2 劳动供给

- 闲暇可以视作一种商品
- 工资上升同时带来替代效应和收入效应
- 因而劳动供给曲线可能向后弯曲

### 7.3 跨期消费

```text
c1 + c2/(1+r) = m1 + m2/(1+r)
```

- 斜率为 `-(1+r)`
- 储蓄者与借款者在利率变化下的福利影响不同

## 8. 消费者剩余

常考内容：

- 保留价格与需求曲线的关系
- 消费者剩余的几何意义
- 价格变化对福利的影响
- CV、EV 与消费者剩余的比较

### 8.1 核心定义

- **保留价格**：消费者愿意为某一单位商品支付的最高价格
- **消费者剩余**：愿付价格与实际支付价格之间的差额总和

### 8.2 常见比较

- 价格下降通常提高消费者剩余
- CV（Compensating Variation）与 EV（Equivalent Variation）是更严格的福利度量
- 消费者剩余常是其近似或图形化表达
