> 北京大学 2026 年春季学期《中级微观经济学》专题整理。内容主要基于课程复习手册与 `Lec5 Endowments`、`Lec6 Consumer's Surplus` 两份 PPT，单独梳理中微后半段里最容易被总复习手册压扁的三块内容。

## 为什么这一段值得单独拆出来

如果把中微前半段理解成“消费者在给定收入和价格下怎么选”，那后半段其实是在问三个更具体的问题：

1. 如果消费者一开始就带着一份资源进入市场，他的选择会怎么变
2. 如果消费可以跨今天和明天重新配置，利率到底在改变什么
3. 如果价格变化了，消费者到底损失或获得了多少福利

这三块内容都和前面的预算约束、最优选择、斯勒茨基分解直接相连，但课堂上经常分别以“禀赋”“跨期”“消费者剩余”三讲出现，所以复习时容易断开。更稳的看法是：它们都是在给“预算约束”和“福利比较”加新的条件。

## 禀赋经济先改写了预算约束

普通两商品模型里，收入 `m` 是一个外生数字；禀赋经济里，收入来自消费者一开始就拥有的商品组合 `(w1, w2)`。

预算约束因此写成：

```text
p1x1 + p2x2 <= p1w1 + p2w2
```

右边的 `p1w1 + p2w2` 是初始禀赋按市场价格折算出来的总价值。这个改写带来两个马上要抓住的几何结论：

- 禀赋点 `(w1, w2)` 一定落在预算线上
- 价格变化时，预算线不是平移，而是绕禀赋点旋转

这就是 `Lec5` 里 “Optimal Consumption with Endowments” 的核心。前半学期你已经习惯把价格变化理解成“绕某个截距旋转”，但在禀赋经济里，真正固定不动的参照点变成了消费者手里原有的资源包。

## 看净需求，而不是只看最终消费

在禀赋经济里，单看 `x1` 已经不够，因为我们更关心消费者是在买还是在卖。

定义净需求：

```text
net demand of good 1 = x1 - w1
```

- `x1 - w1 > 0`：消费者是商品 1 的净需求者，需要到市场上再买
- `x1 - w1 < 0`：消费者是商品 1 的净供给者，会把手里的部分商品卖出去

这个区分很重要，因为同样是 `p1` 上升：

- 对净需求者来说，商品 1 更贵了，通常更不利
- 对净供给者来说，自己手里那部分禀赋也更值钱了，会带来额外收入效应

所以在禀赋模型中，“价格上涨一定让需求下降”这种直觉要更谨慎，它必须重新通过替代效应和收入效应一起判断。

## 含禀赋的斯勒茨基方程比原版多了一层收入来源

普通消费者理论里，斯勒茨基方程把价格变化拆成：

- 替代效应
- 收入效应

到了禀赋经济，这个分解仍然成立，但收入效应里要显式看到“手里已有商品会不会随着价格变化而升值”。

常见写法是：

```text
dx1/dp1 = substitution effect + (w1 - x1) * (dx1/dm)
```

最该记住的不是符号，而是 `w1 - x1` 的经济含义：

- 若 `w1 - x1 > 0`，说明消费者是商品 1 的净供给者
- 这时 `p1` 上升不仅改变相对价格，也会抬高其可支配财富

于是价格效应不再只是“更贵所以少买”，而会混入“我手里这项资产升值了”的成分。

这也是为什么禀赋问题做图时，必须先标出禀赋点，再去画预算线旋转方向；如果一上来只盯着截距，往往会把收入效应判断反。

## 劳动供给题本质上就是一种禀赋模型

把“闲暇”当成一种商品看，中微里常见的劳动供给模型就能直接接到禀赋框架：

- 总时间 `T` 是你的时间禀赋
- 闲暇 `R` 和劳动 `L = T - R` 是这份禀赋的分配方式
- 工资 `w` 是把闲暇换成消费时的机会成本

预算约束可以写成：

```text
C + wR = wT + m0
```

这里：

- `C` 是消费
- `R` 是闲暇
- `m0` 是非劳动收入

工资上升后会同时触发两股力量：

- 替代效应：闲暇变贵，更倾向于少休息、多工作
- 收入效应：更富裕了，如果闲暇是正常品，会更想休息

所以劳动供给可能出现向后弯曲。这不是一个独立章节里的怪结论，而是禀赋模型在“时间资源”这个场景下的直接应用。

## 跨期选择是在两期之间重新分配禀赋

跨期消费常被当成新模型，但它和禀赋经济的关系其实非常紧。

两期模型里，`(m1, m2)` 可以理解为消费者在第一期和第二期的收入禀赋。预算约束写成现值形式：

```text
c1 + c2/(1+r) = m1 + m2/(1+r)
```

或者终值形式：

```text
(1+r)c1 + c2 = (1+r)m1 + m2
```

最关键的图形直觉有两个：

- 预算线斜率是 `-(1+r)`
- 预算线绕 `(m1, m2)` 这个禀赋点旋转

这和前面的禀赋经济完全一致，只不过“商品 1”和“商品 2”现在变成了“今天的消费”和“明天的消费”。

## 利率改变的不是总钱数，而是今天和明天的交换比

`Lec5` 里反复强调的点，是利率变化首先改变相对价格：

- `r` 上升，今天消费的机会成本更高，明天消费相对更便宜
- `r` 下降，今天消费相对更便宜，预算线变平

因此利率上升时，替代效应通常会推动消费者把消费往未来挪。但最终结果还取决于他原本是储蓄者还是借款者。

### 储蓄者

如果原本 `m1 > c1`，说明第一期有结余，会把资源带到第二期。

这时 `r` 上升意味着：

- 储蓄回报更高
- 未来消费更“便宜”
- 同时其资产收益也增加

所以替代效应和收入效应会一起作用，但方向未必总能靠一句话拍死；更稳的做法是结合图形判断其福利通常会改善。

### 借款者

如果原本 `m1 < c1`，说明第一期超前消费，需要借未来的钱。

这时 `r` 上升会让借款更贵，因而：

- 当前消费更难维持
- 福利通常下降

从复习角度看，最重要的是先识别消费者在初始点到底是储蓄者还是借款者，再判断利率变化的后果，而不要把所有比较静态都背成一句统一口号。

## 消费者剩余把“喜欢”翻译成“值多少钱”

消费者剩余这一讲做的事情，是把前面偏好和需求的语言转换成福利衡量的语言。

先定义保留价格（reservation price）：

- 它是消费者愿意为某一单位商品支付的最高价格
- 在边际上，它对应那一单位商品带来的边际效用的货币表达

`Lec6` 的汽油例子就在说明：

- 第一单位商品的保留价格最高
- 数量越多，新增一单位商品的保留价格越低

于是需求曲线可以看成“不同数量对应的边际支付意愿曲线”，消费者剩余则是：

- 需求曲线以下
- 实际价格线以上
- 到所购买数量为止的面积

这让“价格下降让人更高兴”不再只是直觉，而能被近似地量化。

## 为什么保留价格会往下掉

这件事其实就是边际效用递减的另一种说法。

如果消费者已经有了很多单位商品，那么再增加一单位的增量价值通常会下降，所以：

- `r1 > r2 > r3 > ...`

在离散情形下，消费者剩余可以理解成：

```text
(r1 - p) + (r2 - p) + ... + (rn - p)
```

在连续情形下，就变成需求曲线与价格线之间的面积。

因此“保留价格”“需求曲线”“消费者剩余”其实是同一件事的三种表述：

- 保留价格描述单个边际单位值多少钱
- 需求曲线把这些边际支付意愿连成函数
- 消费者剩余把总福利收益加总起来

## CV、EV 和消费者剩余是在比较不同价格下的福利

当价格变化时，单用消费者剩余有时只是近似，所以中微又引入了两个更严格的福利指标：

- `CV`（compensating variation）：价格变化后，要补多少钱才能把消费者拉回原有效用
- `EV`（equivalent variation）：如果想在价格变化前就避免这次变化，消费者最多愿意付多少钱

如果是价格上升：

- `CV > 0`
- `EV > 0`

它们和消费者剩余变化之间通常满足：

```text
EV <= ΔCS <= CV
```

这条关系的直觉是：

- `CV` 用变化后的价格体系衡量“补回去要多少钱”
- `EV` 用变化前的价格体系衡量“事先愿意放弃多少钱”
- `CS` 落在两者之间，作为更容易算的近似量

## 拟线性效用是这一讲最省力的特殊情形

如果效用是拟线性的，收入效应很弱甚至为零，那么消费者剩余就不只是“近似”，而会和更严格的福利测度重合：

```text
EV = ΔCS = CV
```

这也是教材和 PPT 常拿拟线性偏好做例子的原因。它把“面积解释”和“真实福利变化”对齐了，图形和计算都更干净。

## 复习这一段时最值得盯住的抓手

如果考试前只想守住后半门课最关键的骨架，我建议至少保证下面几点能完整说出来：

1. 禀赋模型里的预算线为什么绕禀赋点而不是截距旋转
2. 净需求者和净供给者在价格变化时为什么会表现不同
3. 劳动供给模型为什么本质上是“时间禀赋”的应用
4. 跨期选择里利率改变的是哪一种交换比
5. 保留价格、需求曲线、消费者剩余三者之间是什么关系
6. `CV`、`EV`、`ΔCS` 为什么通常不完全一样

把这些点连起来看，中微后半段就不会只是“几讲零散的新词”，而会重新落回同一条主线上：预算约束如何被改写，福利如何被衡量。
