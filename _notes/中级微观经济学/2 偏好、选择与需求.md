> 北京大学 2026 年春季学期《中级微观经济学》专题整理。内容主要基于课程 PPT 与本地复习手册，单独抽出“偏好-选择-需求-斯勒茨基”这条主线，适合作为总复习手册之外的章节化回看材料。

## 先把这条主线看成一件事

中微前半学期最容易学散的地方，是把每一讲都当成独立名词表：

- 偏好在讲公理
- 效用在讲函数
- 选择在讲切线条件
- 需求在讲曲线
- 显示偏好在讲公理
- 斯勒茨基又突然开始分解效应

其实这些内容连得很紧。

1. 先用偏好描述“消费者想要什么”
2. 再用效用函数给偏好一个便于计算的表示
3. 把偏好和预算约束放在一起，得到最优选择
4. 让价格或收入变化，就得到需求
5. 不直接观察偏好时，可以从实际选择反推显示偏好
6. 价格变动的需求变化，再被拆成替代效应和收入效应

如果把这条线抓住，前半门课就不会只剩下一堆零碎定义。

## 偏好到底在约束什么

偏好关系的三条基本要求是：

- 完备性：任意两个消费束都能比较
- 自反性：每个消费束至少与自己一样好
- 传递性：如果 `x` 至少和 `y` 一样好，`y` 至少和 `z` 一样好，那么 `x` 至少和 `z` 一样好

它们的作用不是“形式主义”，而是为了保证选择行为不会自相矛盾。

无差异曲线部分最值得反复确认两点：

- 两种商品都是好商品时，无差异曲线向右下方倾斜
- 若偏好满足凸性，则边际替代率通常递减

这意味着消费者愿意为了更多的商品 1 放弃多少商品 2，会随着持有组合变化而变化。

## 效用函数只是表示，不是目的

效用函数的关键不是数值有多大，而是能否正确表示排序。

常见结论可以压缩成三句：

- 效用是序数概念，排名比绝对值重要
- 对效用做严格单增变换，不改变偏好
- 真正和选择直接相关的是边际替代率而不是效用值本身

记号上最常出现的是：

```text
MRS = - MU1 / MU2
```

它表示在效用不变时，消费者愿意用多少商品 2 去换一单位商品 1。

几类高频偏好需要一眼认出：

| 类型 | 效用形式 | 题目特征 |
| --- | --- | --- |
| 完全替代品 | `U = ax1 + bx2` | 无差异曲线是直线，常出现角点解 |
| 完全互补品 | `U = min(ax1, bx2)` | 无差异曲线是 L 形，最优点多在折点 |
| 拟线性偏好 | `U = f(x1) + x2` | 收入效应常较弱 |
| 柯布道格拉斯 | `U = x1^a x2^b` | 最常见的内点解模板 |

## 最优选择的核心是“主观替代率 = 市场替代率”

内点解最重要的不是把公式背下来，而是理解为什么会有切线条件。

最优消费束 `(x1*, x2*)` 在内点通常满足：

```text
p1x1* + p2x2* = m
MRS = -p1/p2
```

第二个条件的含义是：

- `MRS` 是消费者主观上愿意交换的比例
- `p1/p2` 是市场客观上必须交换的比例

如果二者不相等，就说明还可以沿预算线移动到更优点。

不同偏好类型下，最优解的形状也不同：

- 柯布道格拉斯通常是内点解
- 完全替代品常在边界上全买一种
- 完全互补品则优先找固定搭配比例

所以做题时，第一步常常不是直接求导，而是先判断偏好的类别。

## 从最优选择到需求

需求不是凭空出现的，它只是“价格或收入变化后，最优选择怎么变”。

这里建议把四类曲线彻底分开：

- 价格提供曲线：某个价格变化时，最优消费束在商品空间中的轨迹
- 普通需求曲线：把上面的轨迹投影到价格-数量平面
- 收入提供曲线：收入变化时，最优消费束的轨迹
- 恩格尔曲线：收入和某种商品需求量之间的关系

这一步最容易混淆的，是把“价格变化”和“收入变化”同时混在脑中。

可以固定使用下面的判断框架：

| 问题 | 先看什么 |
| --- | --- |
| 某商品涨价后需求怎么变 | 价格提供曲线 / 普通需求 |
| 收入增加后买得更多还是更少 | 收入提供曲线 / 恩格尔曲线 |
| 两种商品是替代还是互补 | 交叉价格效应 |

商品分类则至少要分清：

- 正常品：收入上升，需求上升
- 低档品：收入上升，需求下降
- 吉芬商品：价格上升，需求反而上升

其中吉芬商品一定是极特殊情况，不能因为“现实里一起上涨”就乱判。

## 显示偏好是在没有效用函数时倒推偏好

显示偏好这一讲的价值，在于它告诉我们：

即使不知道消费者脑子里的效用函数，也能通过观察选择来排除不一致的偏好结构。

最基本的推理是：

- 如果 `y` 在预算下买得起，但消费者选了 `x`
- 那么 `x` 直接显示偏好于 `y`

再通过传递关系，可以得到间接显示偏好。

这就引出两个常见公理：

- WARP：不允许 `x` 直接显示偏好于 `y` 的同时，`y` 又直接显示偏好于 `x`
- SARP：连直接或间接构成的循环都不允许

这部分常和图形题一起出现。最稳的做法不是死背定义，而是：

1. 先圈出在预算线内可负担的组合
2. 再标出实际选择点
3. 按“被放弃的可负担点”逐步画出显示偏好方向
4. 最后检查是否形成矛盾环

## 斯勒茨基方程把“价格影响需求”拆开

斯勒茨基方程之所以重要，是因为它解释了需求变化的来源。

当价格变化时，消费者行为改变并不只有一个原因：

- 一部分来自相对价格变化，这叫替代效应
- 另一部分来自实际购买力变化，这叫收入效应

标准图形分解可以记成：

1. 原预算线和原最优点
2. 价格变化后的新预算线和新最优点
3. 构造一条补偿预算线，使原购买力被“补回来”
4. 原点到补偿点是替代效应
5. 补偿点到新点是收入效应

变化率形式可以记作：

```text
总效应 = 替代效应 + 收入效应
```

在正常品里，两种效应通常同向推动需求服从需求法则；在低档品里，收入效应可能反向；只有当反向收入效应强到压过替代效应时，才可能出现吉芬现象。

所以“吉芬为什么少见”的根本原因就在这里：它要求很强、很特殊的收入效应结构。

## 复习这块时最值得守住的几个抓手

如果考前时间不多，我会优先确保下面几件事已经清楚：

1. 看到偏好类型时，能快速判断最优解可能是内点还是角点
2. 能解释 `MRS = p1/p2` 背后的经济含义
3. 能分清价格提供曲线、收入提供曲线、需求曲线、恩格尔曲线
4. 能用“可负担但未被选择”判断显示偏好
5. 能完整口述斯勒茨基分解的图形步骤

把这五点连起来，前半学期的大部分题目都会更像同一类题，而不是五六章互不相干的知识碎片。
