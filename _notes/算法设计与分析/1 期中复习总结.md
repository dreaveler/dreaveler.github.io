> 北京大学《算法设计与分析》期中复习整理。内容按题型组织，重点覆盖递推、分治、动态规划、贪心与常见复杂度结论。

## 复习框架

这份总结按考试最常见的答题结构整理：

1. 递推方程求解
2. 分治策略与典型算法
3. 动态规划写法模板
4. 贪心法与正确性证明
5. 回溯、分支限界与平摊分析

## 1. 递推方程

常用方法：

- 迭代法
- 递归树法
- 主定理
- 尝试法

典型结论：

| 递推方程 | 结论 |
| --- | --- |
| `T(n) = T(n-1) + n` | `Theta(n^2)` |
| `T(n) = T(sqrt(n)) + 1` | `Theta(log log n)` |
| `T(n) = 2T(n/2) + n` | `Theta(n log n)` |
| `T(n) = 2T(n/2) + n log n` | `O(n log^2 n)` |
| `T(n) = T(n/3) + T(2n/3) + n` | `O(n log n)` |

主定理使用时，先比较 `f(n)` 与 `n^(log_b a)` 的增长关系，再检查是否满足正则条件。

## 2. 分治策略

分治的标准结构是：

1. 划分子问题
2. 递归求解
3. 合并答案

常见算法与复杂度：

| 算法 | 复杂度 |
| --- | --- |
| 归并排序 | `O(n log n)` |
| 二分查找 | `O(log n)` |
| Karatsuba 整数乘法 | `O(n^(log_2 3))` |
| Strassen 矩阵乘法 | `O(n^(log_2 7))` |
| 最大子段和（分治） | `O(n log n)` |
| 快速排序（均衡） | `Theta(n log n)` |
| 快速排序（最坏） | `Theta(n^2)` |

## 3. 动态规划

动态规划题答题时优先写清：

1. 状态含义
2. 递推方程
3. 边界条件
4. 计算顺序
5. 时间复杂度

常见信号：

- 最优子结构明显
- 有重复子问题
- 题目要求“最优值”“方案数”“最长/最短”

## 4. 贪心法

贪心题不能只写策略，还要补正确性证明。常见证明方式：

- 归纳法
- 交换论证

写法上至少包含：

1. 贪心选择规则
2. 算法过程
3. 正确性证明
4. 复杂度分析

## 5. 其他高频题型

### 回溯与分支限界

- 回溯：核心是解空间树与剪枝条件
- 分支限界：除了约束条件，还要写代价函数或上界/下界

### 平摊分析

常见方法：

- 聚合分析
- 银行家方法
- 势能法

要明确说明采用哪一种方法，并给出每类操作的平摊代价。

## 临考抓手

- 先判断题型，再按该题型的标准答题模板组织答案
- 递推题先选工具，不要硬套主定理
- 动规题优先把状态和转移写对
- 贪心题必须补正确性证明
- 分支限界和平摊分析都要写“为什么这样定义”
