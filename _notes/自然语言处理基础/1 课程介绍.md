## 1. 什么是语言

### 语言的基本定义

词典中的典型关键词可以概括为：

- communication
- system
- word
- human / people

语言既可以指人类自然使用的交流系统，也可以指形式化的符号系统。

### 形式语言与自然语言

| 类型 | 描述 | 示例 |
| --- | --- | --- |
| 形式语言 | 规则严格、语法明确 | C、C++、Java、Python |
| 自然语言 | 人类自然形成并使用 | 中文、英语、日语 |

一个典型思考题是：手语属于形式语言还是自然语言。答案通常偏向自然语言，因为它服务于人类自然交流，并具备完整语言结构。

### 掌握一门语言意味着什么

理解一个简单句子时，实际上会同时调用多个层面：

1. 形态学：词的内部结构
2. 句法：词如何组织成句
3. 词汇语义：词本身的含义
4. 组合语义：句子整体意义如何生成
5. 语用学：语境如何影响理解
6. 语言生成：在上下文中输出恰当表达

这也是 NLP 之所以复杂的根本原因之一。

## 2. 什么是自然语言处理

### 核心定义

自然语言处理（NLP）可以理解为：让计算机能够把人类语言作为输入和或输出。

### 两条主线

#### 自然语言理解 NLU

目标是从自然语言中提取结构化信息，包括：

- 形态结构
- 句法结构
- 语义结构
- 篇章结构
- 与具体应用相关的任务结构

#### 自然语言生成 NLG

目标是从意义表示出发，生成：

- 单词和短语
- 句子
- 段落
- 完整文章

### 常见任务

| 方向 | 代表任务 |
| --- | --- |
| 作为输入 | 语音识别、词性标注、句法分析、信息抽取 |
| 作为输出 | 内容规划、表层生成、文本摘要、语音合成 |
| 复杂任务 | 问答系统、对话系统、机器翻译 |

## 3. 为什么理解语言很难

语言理解的难点主要集中在以下方面：

- 歧义性：一个词或一个句子可能有多种解释
- 上下文依赖：同样的表达在不同场景中含义不同
- 常识推理：很多理解依赖背景知识而非字面信息
- 隐喻与习语：存在大量非字面表达
- 语言演化：词义和用法持续变化

因此，语言理解并不是单纯的字符串匹配，而是结构、语义、知识与上下文的联合推断。

## 4. NLP 的方法演化

课程中给出的主线是：

1. 符号模型：基于规则、逻辑和知识库
2. 统计模型：基于概率和统计学习
3. 神经模型：基于深度学习
4. 预训练语言模型与大语言模型：当前主流范式

可以把整个 NLP 发展过程理解为：从人工规则驱动，逐步走向数据驱动，再走向大规模预训练驱动。
